# Видимость VTC

TrucklineMP позволяет вам контролировать, кто именно может находить вашу виртуальную транспортную компанию и получать к ней доступ. Выберите уровень видимости, который лучше всего соответствует целям вашей VTC.

## Уровни видимости

Каждая VTC имеет настройку видимости, предлагающую три варианта:

| Видимость | В списке каталога | Доступ по ссылке | Кто может просматривать |
|---|---|---|---|
| **Публичная** | Да | Да | Все |
| **Доступна по ссылке** | Нет | Да | Все, у кого есть ссылка |
| **Приватная** | Нет | Нет | Только участники и приглашённые пользователи |

### Публичная

Публичные VTC видны абсолютно всем пользователям TrucklineMP.

- Отображается в каталоге компаний по адресу `trucklinemp.com/vtc`
- Появляется в результатах поиска
- Любой желающий может просматривать профиль
- Любой желающий может подать заявку, если набор открыт
- Оптимально для: устоявшихся VTC, стремящихся к росту и ведущих открытый публичный набор

### Доступна по ссылке

Такие VTC скрыты из каталогов, но доступны по прямой ссылке.

- **Не** отображается в каталоге VTC
- **Не** появляется в результатах поиска
- Любой, у кого есть URL-адрес, может просматривать профиль
- Полезно для: тестирования, предварительного запуска или компаний, предпочитающих органический рост через прямые ссылки

### Приватная

Приватные VTC полностью скрыты, вступление возможно только по приглашениям.

- **Не** отображается в каталоге VTC
- **Не** появляется в результатах поиска
- Профиль доступен для просмотра исключительно участникам
- Новые сотрудники должны вступать через ссылки-приглашения
- Оптимально для: закрытых групп, внутренних команд или VTC, желающих сохранять полный контроль над составом участников

## Как изменить видимость

### Во время создания VTC

При создании новой VTC выберите параметр видимости в форме создания:

1. Перейдите по адресу `trucklinemp.com/vtc/create`
2. Заполните данные вашей компании
3. В блоке **Visibility** выберите вариант Public, Unlisted или Private
4. Завершите процесс создания

### В настройках VTC

Для изменения видимости после создания компании:

1. Перейдите в панель управления вашей VTC
2. Выберите пункт **General Settings**
3. Прокрутите страницу вниз до раздела **Visibility**
4. Выберите вариант Public, Unlisted или Private
5. Нажмите кнопку **Save Configuration**

**Примечание:** Изменение видимости вступает в силу немедленно.

## Приглашения для приватных VTC

Приватные VTC требуют генерации приглашений для вступления новых участников.

### Создание приглашений

1. Перейдите в панель управления вашей VTC
2. Выберите пункт **Invitations** в боковом меню
3. Нажмите кнопку **Create Invite**
4. Настройте параметры приглашения:
   - **Макс. использований** — Сколько раз можно использовать ссылку (оставьте пустым для неограниченного числа)
   - **Истекает через (часы)** — Срок действия ссылки в часах (оставьте пустым, чтобы сделать бессрочной)
   - **ID целевого пользователя (необязательно)** — Ограничить действие приглашения конкретным пользователем
5. Нажмите **Create Invite**
6. Скопируйте ссылку-приглашение и поделитесь ей

### Формат ссылки-приглашения

```
https://trucklinemp.com/vtc/invite/ВАШ_КОД
```

### Управление приглашениями

Страница «Приглашения» отображает все ваши активные и использованные ссылки:

| Статус | Значение |
|---|---|
| **Active** | Ссылку всё ещё можно использовать |
| **Revoked** | Вручную отключено руководством компании |
| **Accepted** | Пользователь вступил в компанию по этой ссылке |
| **Declined** | Пользователь отклонил приглашение |
| **Expired** | Истёк срок действия или исчерпан лимит использований |

### Аннулирование приглашений

Нажмите кнопку **Revoke** напротив любого активного приглашения, чтобы мгновенно его отключить. Отозванные ссылки больше не могут быть использованы.

### Целевые приглашения

Целевые приглашения предназначаются строго конкретному пользователю TrucklineMP:
- Принять такое приглашение может исключительно этот пользователь
- Он должен быть авторизован в системе именно под этой учётной записью
- Удобно для точечного найма конкретных водителей

## Видимость и доступность в поиске

Параметр видимости определяет, кто именно может просматривать профиль вашей VTC. Системный флаг `searchable` контролирует, отображается ли ваша компания в списках каталога и результатах поиска.

- **Public + Searchable** — Отображается везде (по умолчанию)
- **Public + Not Searchable** — Профиль можно просмотреть при наличии ссылки, но в списках он не отображается
- **Unlisted** — Флаг поиска отключается автоматически
- **Private** — Флаг поиска отключается автоматически

В большинстве случаев вам достаточно использовать только базовую настройку видимости.

## Кто и что может просматривать

| Действие | Публичная | По ссылке | Приватная |
|---|---|---|---|
| Просмотр профиля | Любой желающий | Любой при наличии ссылки | Только участники |
| Отображение в каталоге | Да | Нет | Нет |
| Поиск | Да | Нет | Нет |
| Подача заявки | Да (если набор открыт) | Да (если набор открыт) | Нет |
| Вступление по приглашению | Нет | Нет | Да |
| Просмотр мероприятий | Любой желающий | Любой при наличии ссылки | Только участники |
| Просмотр новостей | Любой желающий | Любой при наличии ссылки | Только участники |

## Доступ персонала

Сотрудники администрации TrucklineMP, обладающие разрешением `web.vtc.private.access`, могут просматривать приватные VTC в целях модерации. Это действие не делает их участниками данной компании.

## Рекомендации

### Публичные VTC
- Поддерживайте профиль заполненным и аккуратным
- Включайте рекрутинг, только когда активно ищете новых сотрудников
- Старайтесь обрабатывать заявки оперативно
- Переходите к публичной видимости, когда компания полностью готова к масштабированию

### VTC с доступом по ссылке
- Делитесь вашей ссылкой в целевых узких сообществах
- Используйте этот режим для плавного запуска перед полноценным релизом
- Следите за тем, чтобы настройки найма соответствовали вашей аудитории

### Приватные VTC
- Создавайте целевые приглашения под конкретных водителей
- Устанавливайте разумные сроки действия ссылок ради безопасности
- Отслеживайте список активных приглашений и отзывайте неиспользуемые
- Используйте лимит использований для контроля за распространением ссылки

### Переключение режимов видимости

**Публичная → Приватная:** Действующие участники остаются в компании. Новых сотрудников придётся приглашать вручную.

**Приватная → Публичная:** Любой пользователь сможет найти и просмотреть вашу VTC. Убедитесь, что профиль оформлен должным образом.

**По ссылке → Публичная:** Ваша VTC появится в каталогах и результатах поиска.

## Устранение неполадок

### "Моя VTC не отображается в каталоге"
- Убедитесь, что параметр видимости установлен на **Public**
- Проверьте, включен ли рекрутинг, если вы ищете компанию через фильтры найма
- Подождите несколько минут, пока обновится кэш серверов

### "Люди не могут вступить в мою приватную VTC"
- Убедитесь, что вы создали активное приглашение
- Проверьте, не истёк ли срок действия ссылки и не исчерпан ли лимит её использований
- В случае целевых приглашений убедитесь, что пользователь вошёл в систему под правильным аккаунтом

### "Моя VTC с доступом по ссылке отображается в поиске"
- Такие компании не должны появляться в результатах поиска
- Если это происходит, проверьте, точно ли вы случайно не переключились на режим Public
- Обратитесь в поддержку, если проблема сохраняется

### "Я не могу получить доступ к приватной VTC, куда меня пригласили"
- Убедитесь, что вы авторизованы на сайте
- Проверьте, не истёк ли срок действия ссылки-приглашения
- Если приглашение было целевым, войдите под тем аккаунтом, на который оно отправлялось
