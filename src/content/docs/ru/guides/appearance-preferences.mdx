# Внешний вид и предпочтения

Настройте внешний вид интерфейса TrucklineMP и управляйте параметрами вашей учётной записи.

## Доступ к настройкам внешнего вида

1. Нажмите на свой аватар в правом верхнем углу
2. Выберите пункт **Settings**
3. Перейдите на вкладку **Appearance**
4. Или перейдите напрямую по ссылке `trucklinemp.com/user/me/settings?tab=appearance`

---

## Темы оформления

Выберите, как будет выглядеть TrucklineMP. Ваши предпочтения темы сохраняются автоматически.

### Доступные темы

#### Тёмная
- Эстетика в стиле Marathon по умолчанию
- Тёмный фон со строгим индустриальным дизайном
- Текст с высокой контрастностью
- Не утомляет глаза при слабом освещении

#### Светлая
- Чистый белый фон
- Высокая контрастность для ярко освещённых помещений
- Профессиональный внешний вид
- Легко читается при хорошем дневном свете

#### До боли знакомый серый
- Тёмно-серый цвет с зелёными акцентами
- Уникальная эстетика, которая кажется удивительно знакомой
- Средний уровень контрастности
- Идеальный баланс между тёмной и светлой темами

### Выбор темы
1. Просмотрите карточки тем оформления
2. На каждой карточке отображается палитра с предварительным просмотром цветов темы
3. Нажмите на понравившуюся тему, чтобы применить её
4. Изменения вступают в силу мгновенно и сохраняются автоматически
5. На карточке активной темы появится маленькая точка-индикатор

## Авторизованные приложения

Управляйте сторонними приложениями, которые имеют доступ к вашему аккаунту.

### Доступ к авторизованным приложениям
1. Перейдите в **Settings** > вкладка **Authorized Apps**
2. Или перейдите напрямую по ссылке `trucklinemp.com/user/me/settings?tab=authorized-apps`

### Что такое авторизованные приложения?
Это сторонние приложения, которым вы предоставили доступ к своему аккаунту TrucklineMP через протокол OAuth. К ним могут относиться:
- Конфигураторы грузовиков
- Инструменты для организации мероприятий
- Боты для сообществ
- Аналитические дашборды

### Просмотр приложений
Для каждого авторизованного приложения отображается следующая информация:
- **Название приложения** — Имя стороннего сервиса
- **Предоставленные разрешения** — К каким данным приложение имеет доступ
- **Дата авторизации** — Когда именно вы предоставили доступ

### Отзыв доступа
1. Найдите приложение, доступ которому необходимо прекратить
2. Нажмите кнопку **Revoke**
3. Доступ для этого приложения будет немедленно отозван
4. Если вы захотите снова воспользоваться этим приложением, его придётся авторизовать заново

### Когда следует отзывать доступ
- Вы больше не используете данное приложение
- Вы подозреваете несанкционированный доступ к аккаунту
- Приложение было взломано
- Вы проводите чистку старых и неиспользуемых подключений

## Экспорт данных

Запросите копию ваших персональных данных, хранящихся на серверах TrucklineMP.

### Доступ к экспорту данных
1. Перейдите в **Settings** > вкладка **Data Export**
2. Или перейдите напрямую по ссылке `trucklinemp.com/user/me/settings?tab=export`

### Что входит в экспорт
Ваш архив с данными будет содержать:
- Информацию профиля (имя, идентификатор, аватар, социальные ссылки)
- Историю членства в VTC
- Журналы активности
- Данные конфигураций
- Публикации и созданные темы на форуме
- Записи об участии в мероприятиях

### Как отправить запрос
1. Нажмите кнопку **Request Export**
2. Ваш запрос будет поставлен в очередь на обработку
3. Вы получите электронное письмо, когда архив будет готов
4. Скачайте файл с данными по ссылке из письма
5. Срок действия ссылки на скачивание ограничен по времени

### Формат данных
- Экспортируемые данные предоставляются в формате JSON
- Все данные пригодны для машинного считывания
- Вы можете импортировать их в другие инструменты или сохранить для личного архива

## Настройки форума

Управляйте своими предпочтениями для форума платформы.

### Доступ к настройкам форума
1. Перейдите в **Settings** > вкладка **Forum**
2. Или перейдите напрямую по ссылке `trucklinemp.com/user/me/settings?tab=forum`

### Доступные параметры
- **Настройки уведомлений** — Управление тем, как вы получаете уведомления с форума
- **Параметры отображения** — Настройка внешнего вида контента на форуме
- **Настройки подписи** — Создание и редактирование подписи для ваших сообщений на форуме

## Рекомендации

### Темы оформления
- Выбирайте тему, подходящую под ваше привычное освещение
- Тёмные темы лучше подходят для ночных игровых сессий
- Светлые темы оптимальны для использования днём

### Авторизованные приложения
- Периодически проверяйте список авторизованных приложений
- Отзывайте доступ у сервисов, которыми больше не пользуетесь
- Предоставляйте доступ только тем приложениям, которым доверяете
- Внимательно изучайте список запрашиваемых разрешений перед авторизацией

### Экспорт данных
- Обязательно запрашивайте экспорт данных перед удалением аккаунта
- Храните резервные копии своих данных
- Изучайте экспортированный файл, чтобы понимать, какая информация о вас сохраняется

## Устранение неполадок

### "Тема оформления не переключается"
- Обновите страницу
- Очистите кэш вашего браузера
- Попробуйте открыть сайт в другом браузере

### "Я не могу отозвать доступ у приложения"
- Убедитесь в наличии стабильного интернет-соединения
- Обновите страницу и попробуйте снова
- Если проблема сохраняется, обратитесь в службу поддержки

### "Архив с экспортом данных так и не пришёл"
- Проверьте папку «Спам» в вашей электронной почте
- Экспорт может занять несколько минут
- Обратитесь в поддержку, если письмо не пришло спустя час
