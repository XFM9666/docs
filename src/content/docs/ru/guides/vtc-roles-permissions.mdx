# Роли и разрешения VTC

Роли определяют иерархию вашей VTC и контролируют то, какие действия доступны её участникам. Данное руководство охватывает создание ролей, назначение разрешений и управление общей структурой должностей в компании.

## Доступ к управлению ролями

1. Перейдите в панель управления вашей VTC
2. Выберите пункт **Roles** в боковом меню
3. Вы увидите список всех существующих ролей и подробную информацию о них

---

## Ограничения ролей

| Статус VTC | Лимит ролей |
|---|---|
| Обычная | 20 ролей |
| Верифицированная | Неограниченно |

Если вы достигнете лимита, на экране появится соответствующее предупреждение. Подайте заявку на получение [Статуса верификации](/ru/guides/verified-vtc-program/), чтобы снять любые ограничения на количество ролей.

---

## Создание ролей

1. Нажмите кнопку **Create New Rank** в правом верхнем углу
2. Заполните данные новой роли:
   - **Название роли** (обязательно, максимум 50 символов)
   - **Описание** (необязательно)
   - **Цвет** (выберите цвет для значка роли)
3. Нажмите **Create Role**

Созданная роль сразу же станет доступна для назначения участникам компании.

---

## Редактирование ролей

Нажмите на иконку редактирования на карточке любой роли, чтобы открыть окно настроек. Вы можете управлять параметрами на трёх вкладках:

### Вкладка Детали
- **Название роли** — Изменение отображаемого названия
- **Описание** — Обновление текстового описания роли
- **Цвет** — Изменение цвета значка

Внесённые правки сохраняются при нажатии кнопки **Save Changes**.

### Вкладка Разрешения
Настройте список действий, доступных участникам с этой ролью. Разрешения сгруппированы по категориям:

**Управленческие разрешения:**
- `management.vtc.edit` — Редактирование настроек VTC
- `management.vtc.members` — Управление участниками
- `management.vtc.roles` — Управление ролями
- `management.vtc.recruitment` — Управление рекрутингом
- `management.vtc.news` — Публикация новостей
- `management.vtc.events` — Создание мероприятий

**Другие разрешения:**
- Дополнительные разрешения могут быть доступны в зависимости от конфигурации вашей VTC

Используйте кнопку **Select All** для выдачи всех разрешений в категории или **Deselect All** для их отмены.

### Вкладка Пользователи
Просмотр списка всех участников, обладающих данной ролью:
- Отображение имён сотрудников и их WebID
- Снятие роли с отдельных участников
- Отображение общего числа сотрудников, занимающих эту должность

## Удаление ролей

1. Нажмите на иконку корзины на карточке роли
2. Подтвердите удаление в появившемся модальном окне
3. Участники, обладавшие этой ролью, автоматически её лишатся

**Предупреждение:** Это действие невозможно отменить.

---

## Назначение ролей участникам

Участник может удерживать до **4 ролей** одновременно.

### Со страницы Участники
1. Перейдите в раздел **Members** в боковом меню
2. Нажмите на иконку редактирования рядом с нужным участником
3. Отмечайте или снимайте галочки с ролей в модальном окне
4. Изменения сохраняются автоматически

### Из модального окна редактирования роли
1. Откройте окно редактирования роли
2. Перейдите на вкладку **Users**
3. Используйте интерфейс списка для управления тем, за кем закреплена эта роль

## Рекомендации по иерархии ролей

### Предлагаемая структура
```
Владелец (встроенная роль, удалить невозможно)
├── Администратор
│   ├── Модератор
│   │   ├── Старший водитель
│   │   └── Водитель
```

### Руководство по распределению прав
- **Владелец** — Полный контроль над компанией (встроенная системная роль)
- **Администратор** — Все управленческие разрешения
- **Модератор** — Управление участниками, публикация новостей, создание мероприятий
- **Старший водитель** — Ограниченные разрешения, наставничество
- **Водитель** — Отсутствие специальных разрешений (роль по умолчанию)

### Советы
- Начинайте с простой структуры и добавляйте новые роли только по мере реальной необходимости
- Используйте цвета для визуального разделения уровней иерархии
- Пишите понятные описания для каждой должности
- Регулярно пересматривайте выданные разрешения
- Не перегружайте организационную структуру лишними рангами

## Отображение ролей

### В профилях участников
- Роли отображаются в виде цветных значков
- В профиле участника может быть видимо до 4 значков ролей одновременно
- Значок владельца отображается всегда для владельцев VTC

### В общем списке участников
- Для каждого сотрудника перечисляются все закреплённые за ним роли
- Участники без пользовательских ролей отображаются с пометкой "Unranked" (за исключением владельцев)
- Цвета значков соответствуют цветам, заданным в настройках ролей

## Устранение неполадок

### "Достигнут лимит количества ролей"
- Обычные VTC ограничены лимитом в 20 ролей
- Удалите неиспользуемые роли либо подайте заявку на верификацию компании
- Верифицированные VTC могут создавать неограниченное количество ролей

### "Участники не видят панель управления компании"
- Убедитесь, что им выдано разрешение `management.vtc.edit`
- Проверьте, прикреплено ли данное разрешение к их текущей роли
- Некоторые критические настройки доступны исключительно владельцу VTC

### "Я не могу удалить роль"
- Убедитесь, что данная роль ни за кем не закреплена, либо предварительно переназначайте сотрудников
- Системную роль владельца удалить невозможно
- У вас должно быть разрешение на управление ролями

### "Участник внезапно потерял свою роль"
- Проверьте, не была ли эта роль удалена из системы
- Изучите журнал активности на предмет изменения ролей
- Повторно назначьте роль сотруднику со страницы Участники
