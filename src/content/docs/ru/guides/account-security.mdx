# Безопасность аккаунта

Управляйте безопасностью своей учётной записи, активными сессиями и настройками приватности, чтобы обеспечить надёжную защиту аккаунта.

## Доступ к настройкам безопасности

1. Нажмите на свой аватар в правом верхнем углу
2. Выберите пункт **Settings**
3. Перейдите на вкладку **Security**
4. Или перейдите напрямую по ссылке `trucklinemp.com/user/me/settings?tab=security`

---

## Подключённый аккаунт (Steam)

Ваша учётная запись Steam навсегда привязана к аккаунту TrucklineMP.

- **Статус:** Отключить невозможно
- **Назначение:** Основная верификация личности
- **Steam ID:** Отображается в данном разделе

Эта привязка необходима для подтверждения подлинности аккаунта и не может быть изменена.

## Двухфакторная аутентификация (2FA)

Добавьте дополнительный уровень защиты для вашей учётной записи.

### Включение 2FA

1. Нажмите кнопку **Enable 2FA**
2. Система сгенерирует QR-код
3. Отсканируйте QR-код в вашем приложении-аутентификаторе:
   - Google Authenticator
   - Authy
   - Microsoft Authenticator
   - Любое приложение с поддержкой протокола TOTP
4. Либо вручную введите секретный ключ, указанный под QR-кодом
5. Нажмите **Continue to Verification**
6. Введите 6-значный код из вашего приложения
7. Нажмите **Verify & Enable**
8. **Сохраните резервные коды** — они больше никогда не будут показаны снова

### Резервные коды

После включения 2FA вы получите **10 резервных кодов**:
- Каждый код можно использовать **ровно один раз**
- Используйте их, если вы потеряете доступ к приложению-аутентификатору
- Храните их в надёжном месте (менеджер паролей, распечатанная копия)
- Вы можете скопировать все коды одновременно нажатием одной кнопки

### Отключение 2FA

1. Нажмите кнопку **Disable 2FA**
2. Введите код из приложения-аутентификатора **или** один из резервных кодов
3. Нажмите **Disable 2FA**
4. Двухфакторная защита удалена с вашего аккаунта

**Предупреждение:** Отключение 2FA существенно снижает уровень безопасности вашей учётной записи.

### Если вы потеряли доступ к аутентификатору

1. Войдите в аккаунт, используя один из резервных кодов
2. Выполнив вход, перейдите в Настройки безопасности
3. Отключите 2FA с помощью резервного кода
4. Повторно включите 2FA, привязав новое приложение-аутентификатор

**Если резервных кодов не осталось:** Обратитесь в службу поддержки через Discord, предоставив доказательства владения аккаунтом.

## Режим разработчика

Включите Консоль разработчика для доступа к профессиональным инструментам платформы.

### Что вы получаете
- Создание API-проектов
- Управление учётными данными API
- Доступ к документации разработчика
- Создание собственных интеграций с TrucklineMP

### Включение режима разработчика
1. Нажмите кнопку **Activate** в блоке Developer Mode
2. Внимательно прочитайте диалоговое окно подтверждения
3. Нажмите **Confirm Activation**
4. Режим разработчика теперь активен

**Предупреждение:** Это действие является окончательным и необратимым.

### Доступ к Консоли
- После включения нажмите кнопку **Open Console** для перехода к инструментам разработчика
- Или перейдите напрямую по ссылке `trucklinemp.com/developer`

## Активные сессии

Просматривайте и управляйте всеми устройствами, на которых в данный момент выполнен вход в ваш аккаунт.

### Информация о сессии
Для каждой сессии отображается:
- **Браузер** — Chrome, Firefox, Safari, Edge
- **Операционная система** — Windows, macOS, Linux, Android, iOS
- **IP-адрес** — IP-адрес, с которого был выполнен вход
- **Дата начала** — Время создания сессии
- **Плашка "Current"** — Отмечает вашу текущую сессию

### Отзыв (завершение) сессии
1. Найдите сессию, которую необходимо завершить
2. Нажмите кнопку **Revoke** рядом с ней
3. Выбранный сеанс немедленно завершится
4. На этом устройстве потребуется выполнить повторный вход

### Завершение всех остальных сессий
1. Нажмите кнопку **Sign out all other sessions** в верхней части блока
2. Все сессии, кроме вашей текущей, будут мгновенно завершены
3. Полезно, если вы подозреваете несанкционированный доступ к аккаунту

## Настройки приватности

### Скрыть внешние конфигурации
- Когда включено, ваши конфигурации Truckline Utils видны только вам и администраторам платформы
- Другие пользователи не смогут просматривать эти конфигурации в вашем профиле
- Установите флажок, чтобы включить или отключить параметр

## Рекомендации по безопасности

### Обязательно
- **Включите 2FA** — Самый важный шаг для защиты учётной записи
- **Сохраните резервные коды** — Поместите их в надёжный менеджер паролей
- **Регулярно проверяйте сессии** — Завершайте любые сеансы, которые вам незнакомы
- **Используйте надёжный пароль** — Он должен быть уникальным для TrucklineMP

### Рекомендуется
- Всегда выходите из аккаунта на привязанных устройствах
- Периодически проверяйте список авторизованных приложений
- Поддерживайте актуальность привязанных аккаунтов
- Немедленно сообщайте о любой подозрительной активности

## Устранение неполадок

### "Я не могу включить 2FA"
- Убедитесь, что ваше приложение-аутентификатор поддерживает протокол TOTP
- Проверьте, точно ли синхронизировано системное время на вашем телефоне/устройстве
- Попробуйте использовать другое приложение-аутентификатор

### "Я потерял свои резервные коды"
- Если вы всё ещё авторизованы в системе, отключите 2FA и включите её заново
- Если доступ полностью потерян, свяжитесь с поддержкой через Discord
- Предоставьте доказательства личности для восстановления учётной записи

### "Я вижу сессию, которую не узнаю"
- Немедленно нажмите **Revoke** напротив этой сессии
- По возможности сразу измените свой пароль
- Включите 2FA, если она ещё не активирована
- Напишите в поддержку, если подозреваете взлом

### "Я не могу войти в систему с помощью кода 2FA"
- Проверьте правильность времени в приложении-аутентификаторе (синхронизацию часов)
- Воспользуйтесь одним из резервных кодов
- Если ничего не помогает, обратитесь в службу поддержки
