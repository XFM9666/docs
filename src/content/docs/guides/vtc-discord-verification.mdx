# VTC Discord Verification

Link your VTC's Discord server to TrucklineMP to display it on your public profile and build trust with potential members.

## Accessing Discord Settings

1. Navigate to your VTC's management panel
2. Select **General Settings** from the sidebar
3. Scroll to the **Discord Server** section

---

## Why Verify Your Discord?

- **Build Trust** — Show applicants your Discord is legitimate
- **Increase Engagement** — Make it easy for members to find your server
- **Professional Appearance** — Displayed prominently on your VTC profile
- **Community Building** — Connect your TrucklineMP and Discord communities

---

## Verification Process

### Step 1: Generate Token
1. Click **Generate Verification Token**
2. A unique token is created for your VTC
3. The token is displayed on screen
4. Copy the token using the copy button

### Step 2: Invite the Bot
1. Click the **Invite the TrucklineMP bot** link
2. Select your Discord server from the dropdown
3. Grant the bot necessary permissions
4. Complete the invitation

**Required Bot Permissions:**
- Read Messages/View Channels
- Send Messages
- Use Slash Commands

### Step 3: Verify the Server
1. Go to any channel in your Discord server
2. Type the command: `/verify-server <TOKEN>`
3. Replace `<TOKEN>` with your actual token
4. The bot will confirm the verification
5. Your server is now linked

### Step 4: Confirmation
- Return to TrucklineMP
- The page will automatically detect the verification
- You'll see a success notification
- Your Discord server is now displayed on your VTC profile

---

## Token Information

### Token Expiration
- Tokens expire after **15 minutes**
- If your token expires, generate a new one
- Each VTC can only have one active token at a time

### Token Security
- Tokens are unique to your VTC
- Don't share tokens publicly
- Generate a new token if compromised

---

## Server Visibility

Control who can see your linked Discord server on your VTC profile.

### Visibility Options

**Only You**
- Only the VTC owner can see the Discord server
- Hidden from members and the public
- Useful during setup or if the server is private

**VTC Members**
- Visible to all members of your VTC
- Hidden from the public
- Good for member-only communities

**Everyone**
- Visible on your public VTC profile
- Anyone can see your Discord server
- Recommended for public recruitment

### Changing Visibility
1. Go to the Discord Server section
2. Click the desired visibility option
3. The change is saved automatically
4. You'll see a success notification

## Managing Your Linked Server

### Viewing Server Status
- See your linked server name
- Check verification status (Verified/Unverified)
- View current visibility setting

### Generating a New Token
- Click **Generate New Token**
- The old token is invalidated
- Use the new token to re-verify if needed

### Unlinking Your Server
1. Click **Unlink Server**
2. Confirm the action
3. Your Discord server is removed from your profile
4. You can link a different server later

## Troubleshooting

### "The bot isn't responding to my command"
- Ensure the bot has permission to read messages in the channel
- Check that you're using the correct token
- Verify the bot is in your server
- Try a different channel

### "My token expired"
- Tokens expire after 15 minutes
- Generate a new token
- Use the new token immediately

### "The verification isn't showing on my profile"
- Refresh the page
- Check that verification completed successfully
- Ensure visibility is set to "Everyone" or "VTC Members"

### "I want to link a different server"
- Unlink your current server first
- Generate a new token
- Follow the verification process for the new server

### "The bot doesn't have permission"
- Go to your Discord server settings
- Navigate to Roles
- Ensure the TrucklineMP bot has the required permissions
- Try the command again

## Best Practices

### Server Setup
- Use a permanent invite link
- Set up clear channels and rules
- Have moderators ready to welcome new members
- Keep your server active and engaging

### Visibility
- Set visibility to "Everyone" for public recruitment
- Use "VTC Members" for member-only communities
- Use "Only You" during initial setup

### Maintenance
- Check your Discord server regularly
- Respond to member questions promptly
- Keep your server organized
- Update your VTC profile when Discord changes

---

## Requirements

To verify your Discord server, you need:
- **Manage Server** permission in the Discord server
- **Management permissions** in your VTC on TrucklineMP
- A Discord server that meets Discord's guidelines
